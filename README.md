# AI-Powered Face Mask Detection 😷

## Overview
This project implements a deep learning model to detect whether individuals are wearing a face mask in real-time using computer vision techniques. It leverages Convolutional Neural Networks (CNNs) for image classification and can be integrated into surveillance systems, mobile apps, or IoT devices.

## Features
- Real-time detection using webcam or video streams
- High accuracy with pre-trained deep learning models
- Easy integration into existing applications
- Dataset support for retraining and fine-tuning

## Tech Stack
- Python
- TensorFlow/Keras
- OpenCV
- NumPy

## Dataset
The model is trained on a labeled dataset containing images of people with masks and without masks. You can expand the dataset for better generalization.
